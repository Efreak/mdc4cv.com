    Depletion - Depth Calculator

MDC Depletion Depth Calculator

Voltage = q/(2*k*eo) * Doping * (Depletion Width)^2 - Vbi

Volts   =q/2*k*eo     cm^-3   microns ^2 - Vbi

Enter values for two of the three parameters and set the third value equal to 0.

The 0 value parameter will be computed when you click the Calculate button.

Valid for silicon.
