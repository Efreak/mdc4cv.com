MDC FPP SOFTWARE

 

For use with a manual four point prober, the MDC FPP Software operates in a convenient, single screen that displays both measurement parameters and test results.  The FPP software can measure Resistivity, Conductivity, Resistance, Doping, Thickness, and Sheet Resistance when used with a compatible current source and voltmeter or SMU.  

The software uses standard NIST or SEMI test methods and implements corrections based on probe spacing, sample thickness, diameter, and temperature.  The MDC program automatically selects the best test current to minimize sample heating and tests both polarities. This software may be used with various current-voltage meters that operate in the four-terminal Kelvin mode.
